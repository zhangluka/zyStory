## 背景

变更需交付一张结婚单身夜派对邀请海报的 HTML 单页，由 Bobby & Yang 发起邀约；参考图为手绘极简风格（米白底、黑色线条、粉/红点缀、迪斯科球与手绘边框）。当前无既有海报或共享组件，从零实现单页即可。

## 目标 / 非目标

**目标：**

- 交付一个自包含的 HTML 文件，打开即可展示完整海报。
- 信息区：主人公、邀约语、时间/地点/别墅名（占位）、活动概览，均为中文；主题句沿用图二风格（中英）。
- 视觉：手绘极简、米白背景、黑线+少量粉/红、可选迪斯科球与波浪边框；适配手机竖屏与打印。
- 时间/地点/别墅名为占位文案，便于后续全局替换。

**非目标：**

- 不做后端、表单提交、多语言切换或动态数据。
- 不做构建链（如 npm/Vite）；仅静态 HTML（可含内联或同一目录下 CSS）。
- 不做复杂交互（如动画、弹窗），仅静态展示。

## 决策

- **单文件 HTML**：所有样式内联或通过单文件 `<link>` 引用，避免多资源路径带来的分享与部署问题；便于微信/邮件转发与本地打开。
- **手绘感实现**：边框与简单图形用 CSS 描边或内联 SVG；迪斯科球等复杂图形优先用内联 SVG 或 data-URI 小图，避免外部图片依赖；若需更精细插画可后续替换为 PNG/SVG 文件。
- **字体**：中文使用系统易读字体（如 `"PingFang SC"`, `"Microsoft YaHei"`, `sans-serif`），标题可选用 Google Fonts 手写体（如 Caveat）仅作英文点缀；通过 `@import` 或 `<link>` 引入，并设置 `font-display: swap` 与合理 fallback。
- **配色**：背景约 `#F5F0E8`（米白），正文/线条 `#1a1a1a`，点缀色 `#c45c5c` / `#e8a0a0`，与参考图一致。
- **布局与响应式**：单列、从上到下分块（标题 → 插画区 → 信息块 → 主题句 → 落款）；`max-width` 控制阅读宽度，`viewport` meta 与相对单位保证手机竖屏可读；打印时通过 `@media print` 去掉多余边距、保证一页内完整展示。

## 风险 / 权衡

- **手绘素材质量**：纯 CSS/SVG 难以完全复刻参考图的手绘质感 → 先实现几何化/简化版，后续可替换为设计师提供的 SVG/PNG。
- **字体加载**：外链字体在离线或弱网下可能缺失 → 始终设置通用 fallback，重要信息不依赖单一字体。
- **打印尺寸**：不同设备默认打印尺寸不一 → 在 CSS 中建议 `@media print` 下的尺寸与边距，用户仍可在打印对话框中调整比例。
